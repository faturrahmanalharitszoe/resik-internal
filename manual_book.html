Buku Panduan Pengguna (User Manual) - Resik Internal

Selamat datang di aplikasi Resik Internal. Aplikasi ini dirancang untuk memudahkan kolaborasi dan komunikasi di dalam perusahaan melalui tiga fitur utama: Chat, Notion (Pusat Catatan & Kolaborasi), dan Folder Bersama.

Catatan: Panduan ini menggunakan blok penanda untuk tempat gambar (placeholder). Harap hapus kotak placeholder tersebut dan tambahkan gambar yang sesuai (Insert > Pictures) saat mengedit di Microsoft Word.

1. Fitur Chat (Percakapan)

Fitur Chat memungkinkan Anda untuk berkomunikasi dengan rekan kerja secara real-time, baik melalui grup (ruangan) maupun pesan pribadi (Direct Message).

1.1. Mengakses Chat

  • Klik ikon Chat (💬) pada panel navigasi di sebelah kiri (Activity Bar).
  • Anda akan melihat daftar Ruangan (Rooms) dan daftar Pengguna Online.
[Tambahkan Screenshot: Tampilan utama daftar Chat dan pengguna online di sini]

1.2. Membuat Ruangan Baru (Grup)

  1. Pada sidebar sebelah kiri, cari bagian 💬 Ruangan.
  2. Klik tombol + (Buat Ruangan Baru) di sebelah kanan label.
  3. Masukkan nama ruangan (contoh: tim-marketing).
  4. Klik tombol Buat.
[Tambahkan Screenshot: Modal pembuatan ruangan baru di sini]

1.3. Mengirim Pesan

  1. Pilih ruangan atau nama pengguna dari sidebar.
  2. Ketik pesan Anda di kotak input bagian bawah.
  3. Klik tombol Kirim (ikon panah) atau tekan tombol Enter.

1.4. Manajemen Anggota Ruangan

  • Untuk melihat daftar anggota ruangan, klik ikon Anggota (👥) di pojok kanan atas area chat.
  • Untuk menambahkan anggota baru, buka panel anggota lalu klik tombol + Tambah Anggota.
[Tambahkan Screenshot: Panel anggota ruangan dan modal penambahan anggota di sini]

2. Fitur Folder Bersama (Manajemen Dokumen)

Fitur Folder Bersama digunakan untuk mendokumentasikan dan melacak surat/dokumen masuk dan keluar antar divisi atau proyek.

2.1. Mengakses Folder Bersama

  • Klik ikon Folder (📁) pada navigasi kiri (Activity Bar) atau klik menu Folder Bersama di sidebar.
[Tambahkan Screenshot: Tampilan dasbor Folder Bersama beserta metrik statistik di sini]

2.2. Mengunggah Dokumen Baru

  1. Pada halaman Folder Bersama, klik tombol Unggah Dokumen di bagian atas (banner).
  2. Akan muncul form pengisian, lengkapi data berikut:
    • Tipe Dokumen & Sub-Tipe (contoh: PO Marketing)
    • Proyek
    • Nomor & Nama Dokumen
    • Tanggal Dokumen
    • Deskripsi (opsional)
    • Penerima Dokumen
  3. Klik pada area Berkas File (📁) atau seret file (drag and drop) ke area tersebut (format yang didukung: .pdf, .doc, .docx, .xls, .xlsx).
  4. Klik Upload Dokumen.
[Tambahkan Screenshot: Form Upload Dokumen Baru di sini]

2.3. Memantau Dokumen Masuk dan Keluar

  • Di halaman utama Folder Bersama, terdapat tab Masuk dan Keluar.
  • Masuk: Menampilkan dokumen yang ditujukan ke divisi/pengguna Anda.
  • Keluar: Menampilkan dokumen yang Anda unggah/kirimkan.
  • Gunakan Kotak Pencarian atau filter Bulan & Tahun untuk mempermudah pencarian dokumen spesifik.
[Tambahkan Screenshot: Tabel dokumen masuk dan keluar dengan filter di sini]

3. Fitur Notion (Pusat Catatan & Kolaborasi)

Fitur Notion berfungsi sebagai kanvas digital untuk mencatat, membuat dokumentasi, dan melacak pekerjaan dengan berbagai tampilan interaktif (Database, Board, Calendar, dll).

3.1. Mengakses Workspace

  • Klik ikon Notion (📄) pada panel navigasi (Activity Bar).
[Tambahkan Screenshot: Tampilan Halaman Utama Notion Workspace di sini]

3.2. Membuat dan Mengedit Halaman

  1. Untuk membuat halaman (sub-halaman) baru, klik tombol + Halaman Baru di pojok kanan atas.
  2. Untuk mengubah konten halaman yang sedang aktif, klik tombol Edit.
  3. Saat mode edit aktif, gunakan toolbar untuk memformat teks:
    • H untuk Heading (Judul)
    • • untuk Bullet List
    • B untuk huruf tebal (Bold)
    • I untuk huruf miring (Italic)
  4. Klik Simpan setelah selesai.
[Tambahkan Screenshot: Tampilan Mode Edit Halaman Notion di sini]

3.3. Mengganti Ikon dan Cover Halaman

  • Ikon Halaman: Klik ikon emoji di sebelah kiri judul halaman (contoh: 📄) untuk memilih emoji baru dari menu pop-up.
  • Cover Halaman: Klik tombol Ganti Cover di sudut kanan atas area header (gradien warna) untuk memperbarui latar belakang header halaman.

3.4. Menggunakan Mode Database

Notion Workspace mendukung manajemen tugas/data yang terstruktur jika Anda menggunakan halaman bertipe Database.

  • Ubah tampilan data menggunakan tab: Table, Board, Calendar, Gallery, List, atau Timeline.
  • Gunakan menu Filter (Status, Priority, Assignee) untuk menyaring data yang ditampilkan.
  • Klik + Baris Baru untuk menambahkan item tugas/data baru ke dalam database.
[Tambahkan Screenshot: Tampilan Database dengan Tab Board/Table dan Filter aktif di sini]
Tip: Navigasi antar fitur sangat mudah, cukup gunakan tombol-tombol utama (Chat, Notion, Folder) di panel Activity Bar (sebelah paling kiri layar) kapan pun Anda membutuhkannya.
